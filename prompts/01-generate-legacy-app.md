# Step 1: レガシーアプリ生成 — プロンプト例

> **事前に確認してください:**
> - `.github/copilot-instructions.md` — プロジェクト全体の背景と共通規約
> - `.github/instructions/step1-generate-legacy.md` — このステップ用の詳細ガイドライン

このステップでは、Copilot Chat を使ってレガシー言語のアプリケーションを自動生成します。
言語やアプリの題材は **自由に指定できます**。以下を参考に、自身の選択で Copilot に指示を出してください。

---

## テンプレート（カスタマイズ可能）

```
[レガシー言語名] で [アプリケーション題材] を作成してください。

以下の機能を含めてください：
- [機能 1]
- [機能 2]
- [機能 3]

ソースコードとデータファイルは workspace/legacy/ に配置してください。
```

---

## 具体例

### 例 1: COBOL × 在庫管理（サンプル）

```
COBOL で在庫管理システムを作成してください。
以下の機能を含めてください：
- ファイルからの在庫データ読み込み
- 入庫・出庫処理（在庫の増減）
- 在庫不足アラート（閾値以下の商品を検出）
- 在庫レポートの出力

ソースコードとデータファイルは workspace/legacy/ に配置してください。
```

### 例 2: VB6 × 給与計算

```
VB6 で給与計算バッチシステムを作成してください。
以下の機能を含めてください：
- CSV ファイルからの従業員データ読み込み
- 基本給・手当・控除の計算
- 所得税の簡易計算
- 給与明細レポートの出力

ソースコードとデータファイルは workspace/legacy/ に配置してください。
```

### 例 3: Perl × 顧客管理

```
Perl で顧客管理システムを作成してください。
以下の機能を含めてください：
- テキストファイルからの顧客データ読み込み
- 顧客の検索（名前・ID）
- 顧客情報の追加・更新
- 顧客一覧レポートの出力

ソースコードとデータファイルは workspace/legacy/ に配置してください。
```
