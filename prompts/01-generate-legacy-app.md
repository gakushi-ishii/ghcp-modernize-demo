# Step 1: レガシーアプリ生成 — プロンプト例

このステップでは、Copilot Chat を使ってレガシー言語のアプリケーションを自動生成します。
言語やアプリの題材は **自由に指定できます**。以下を参考に、自身の選択で Copilot に指示を出してください。

> 💡 **Step 2（モダナイズ）だけ試したい場合**は、このステップをスキップして `workspace/legacy/example/` のサンプル（VBScript 在庫管理）を使用できます。

---

## テンプレート（カスタマイズ可能）

```
[レガシー言語名] で [アプリケーション題材] を作成してください。

以下の機能を含めてください：
- [機能 1]
- [機能 2]
- [機能 3]

ソースコードとデータファイルは workspace/legacy/work/ に配置してください。
```

---

## 具体例

### 例 1: VBScript × 在庫管理

```
VBScript で在庫管理システムを作成してください。
以下の機能を含めてください：
- ファイルからの在庫データ読み込み
- 入庫・出庫処理（在庫の増減）
- 在庫不足アラート（閾値以下の商品を検出）
- 在庫レポートの出力

ソースコードとデータファイルは workspace/legacy/work/ に配置してください。
```

### 例 2: VBScript × 給与計算

```
VBScript で給与計算バッチシステムを作成してください。
以下の機能を含めてください：
- CSV ファイルからの従業員データ読み込み
- 基本給・手当・控除の計算
- 所得税の簡易計算
- 給与明細レポートの出力

ソースコードとデータファイルは workspace/legacy/work/ に配置してください。
```

### 例 3: Batch Script × 顧客管理

```
Batch Script で顧客管理システムを作成してください。
以下の機能を含めてください：
- テキストファイルからの顧客データ読み込み
- 顧客の検索（名前・ID）
- 顧客情報の追加・更新
- 顧客一覧レポートの出力

ソースコードとデータファイルは workspace/legacy/work/ に配置してください。
```

## 確認ポイント

生成後、以下を確認してください：

- [ ] `workspace/legacy/work/docs/how-to-run.md` が生成されているか
- [ ] `workspace/legacy/work/docs/specification.md` が生成されているか
- [ ] `workspace/legacy/work/data/` にサンプルデータが含まれているか
- [ ] ソースコードに日本語コメントが含まれているか
- [ ] 動作確認ができているか（言語別の対応は [README.md](../README.md#動作確認環境について) を参照）
